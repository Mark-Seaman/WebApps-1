# Lesson 3 - Github Repos

### Step 1 - Create Github Account
* Visit the Github website at [https://github.com](https://github.com)
* Select a name that will be your permanent Github user name
* Sign up for a free account at Github
* Login to your account


### Step 2 - Create Github Repo

Setup a git repository stored at Github that will be used for all the code you
will write for this course. All grading will be done by examining the code in this
course repo.

* Visit my Git Repo
    * Github URL: **[https://github.com/Mark-Seaman/WebApps-1](https://github.com/Mark-Seaman/WebApps-1)**
* Fork my repo to create a Github repo for your code.  
* Now you have your own version of the code. Put all your code changes here.


### Step 3 - Clone Your Git Repo

* Run Visual Studio Code
* If you are missing git, install it from "https://git-scm.com/downloads".
* Clone repo from Github
* Set permissions to access Github account from VS Code
